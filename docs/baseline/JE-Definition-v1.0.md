# 判断工程（Judgment Engineering）定义

## 一、标准定义

判断工程不是让 AI 更聪明。

判断工程是让系统拥有：

是否允许 AI 继续运行的最终裁定权。

AI 负责生成判断意见。  
Agent 负责执行流程。  
系统负责决定：

- 是否继续判断
- 是否停止判断
- 是否进入执行

---

## 二、边界声明

判断工程不：

- 提升模型推理能力
- 替代 AI 判断
- 替代 Agent 执行
- 保证判断正确性

判断工程只处理：

判断流是否允许继续。

---

## 三、适用场景

当输出开始承担以下任一条件时，判断工程成为必要结构：

- 不可逆后果
- 责任归属要求
- 组织级复用
- 执行触发
