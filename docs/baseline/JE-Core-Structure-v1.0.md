# 核心结构模型

## 一、结构关系

AI → 判断流 → 判断工程层 → 
    ├─ 继续
    ├─ 停止
    └─ 执行 → Agent

---

## 二、权力分离原则

AI：

- 生成判断
- 无裁定权

Agent：

- 执行流程
- 无裁定权

系统：

- 拥有裁定权
- 控制判断流运行状态

---

## 三、最小构成单元

判断工程包含四个对象：

1. 判断流
2. 裁定权
3. 裁定事件
4. 状态切换

缺失任一项，裁定结构失效。
